# DiceGameTest Project Instructions

## Project Overview
Browser-based dice board game with RPG elements. Two versions:
- **V1**: Single-player classic (active development)
- **V2**: Type system with Fire/Water/Grass (on hold)

## Important Rules
- **Only modify V1** unless explicitly asked otherwise
- Test in incognito mode (no need to share links)
- Push to GitHub when changes are ready for testing

## File Structure
```
v1/
  index.html  - Main HTML structure
  styles.css  - All styling including mobile responsive
  game.js     - All game logic
```

## Key Configs in game.js (top of file)
- `TIMING` - All timing/speed values (combat, dice, movement)
- `TEST_MODE` - Enable test mode and set tile types
